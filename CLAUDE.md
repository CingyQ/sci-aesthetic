# 科研绘图指南 - 项目文档

## 项目概述
单文件离线交互式教学网站，教授科研绘图与PPT设计技巧。面向科研工作者，以高标准教学内容和丰富交互组件为特色，覆盖从色彩理论到出版级可视化的完整工作流。

## 技术栈
- 单一 HTML 文件（科研审美.html），无需服务器
- CSS（内联），JS（内联），SVG（内联）
- 字体：Playfair Display + Raleway + Noto Sans SC + JetBrains Mono（Google Fonts）

## 配色系统
- 主色：天蓝 #7EC8E3 / 浅 #D0EDF7 / 深 #5BA3C9
- 辅色：薄荷绿 #95D5B2 / 浅 #D8F3DC / 深 #74C69D
- 三级色：薰衣草 #B8B8E8 / 浅 #E0E0F5 / 深 #8A8AC8
- 背景：极淡蓝白 #FAFCFD / #F3F7F9 / #EBF0F3
- 文字：深蓝灰 #2C3E50 / #5D6D7E / #95A5A6
- 边框：#E0E8ED

- p21: 出版级图表 - 主题定制器 + 期刊投稿图片规格表 + ggsave代码生成器 + patchwork多面板布局
- p22: PPT排版字体（拖拽式布局编辑器）

## 教学标准
- 每个页面需有明确的教学目标和可操作的交互练习
- 理论讲解配合实际案例，避免空洞描述
- 交互组件需提供即时反馈，增强学习效果
- 专业术语使用中文+英文标注，兼顾不同背景读者
- 内容深度需达到研究生/博士生科研绘图实际需求

## 开发规范
- 所有文字使用中文（技术术语括号注英文）
- 所有交互必须完整可用
- SVG图表使用内联方式
- 懒加载初始化（页面切换时才初始化对应组件）
- 配色系统严格遵循 design-spec.md 中的变量定义
