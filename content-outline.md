## 模块一：科研数据可视化（10 页）

> 从理解色彩到制作出版级图表的完整链路。
> 色彩理论 → 配色实践 → 图表选择 → R/Python 编程 → 导出发表。

### m1-p1 色彩理论基础
**教学目标**：理解光与色彩的物理/感知基础。
- 交互色轮（Canvas，拖拽选色，实时显示 HEX/RGB/HSL/CMYK）
- HSL 三维度独立控制演示（3 个滑块联动色轮 + 色块）
- 色彩空间对比演示（RGB 立方体 vs HSL 圆柱 — D3 3D 简图）
- 加色混色 vs 减色混色交互（叠加圆形，拖动看混色结果）

### m1-p2 色彩和谐与科研配色
**教学目标**：掌握经典配色方案的原理，直接获取可用的专业配色。
- **色彩和谐理论**：
  - 互补/类似/三角/分裂互补/方形 5 种方案（色轮上实时标记 + 图表应用预览）
  - 60-30-10 配色法则交互演示（拖拽调整比例，实时预览图表效果）
  - 色温与情绪：暖色/冷色/中性色在科研图表中的心理暗示
  - 色距理论（CIELAB deltaE 计算器——输入两色计算感知差异）
- **科研配色方案库**：
  - 配色方案浏览器：按来源分类（Nature/Science/Lancet/JAMA/Cell/PNAS/ggsci/ColorBrewer）
  - 每种方案：色板展示 + 一键复制 + 应用到 4 种示例图表的实时预览
  - Rainbow 陷阱 vs 感知均匀配色对比演示（同一数据用 rainbow vs viridis，拖拽对比）
  - Wong 色盲安全 8 色方案 + 使用场景

### m1-p3 配色生成器与数据配色
**教学目标**：动手创建自定义配色，理解三类数据配色的选择原则。
- **自定义配色生成器**：
  - 输入基色 → 选择算法（互补/类似/三角/单色渐变） → 生成方案
  - 实时预览：生成的配色应用到柱状图/折线图/热力图
  - 微调面板：每个色可独立调整 HSL
  - 导出：HEX 列表 / RGB / R scale_color_manual / Python matplotlib / CSS 变量
- **三类数据配色**：
  - 连续（Sequential）：适合单变量递增数据 → 热力图演示
  - 发散（Diverging）：适合有中心值的数据 → 相关矩阵演示
  - 定性（Qualitative）：适合分类数据 → 散点分组演示
  - 配色选择器：每类 5 种子方案，颜色数量滑块（3-12）
  - "错误 vs 正确"对比：常见的配色错误 + 修正建议

### m1-p4 色彩无障碍
**教学目标**：确保图表对色觉障碍者友好。
- WCAG 对比度检测器（前景/背景输入 → AA/AAA 判定 + 具体比值）
- 色觉缺陷类型教学（红绿色盲/蓝黄色盲/全色盲，附患病率统计 D3 图表）
- 自定义配色色盲模拟器（输入 N 个颜色 → 3 种色盲视角并排预览 → 安全/危险判定）
- 修复建议：不安全的配色自动推荐安全替代方案

### m1-p5 图表选择指南
**教学目标**：面对数据时，快速判断用哪种图表。
- 交互式决策树：你的数据是什么类型？→ 变量几个？→ 要表达什么关系？→ 推荐图表
- 图表类型全览（20+ 种，按用途分组：比较/分布/关系/组成/趋势）
- 每种图表的适用/不适用场景 + 好例/坏例对比
- "我应该用饼图吗？"常见误用专题

### m1-p6 ggplot2 图层语法与分面
**教学目标**：理解 ggplot2 的核心设计哲学与高级布局控制。
- **图层语法**：
  - 图层叠加交互演示：从空白画布逐层添加（data → aes → geom → scale → theme）
  - 每加一层，SVG 预览实时更新 + R 代码同步高亮新增行
  - 粘性滚动教学：左侧代码固定，右侧图表随滚动逐步变化
- **分面与坐标系统**：
  - Faceting 分面演示：none → facet_wrap → facet_grid（3 个 tab，SVG + R 代码同步切换）
  - 坐标变换：笛卡尔 → flip → polar（同一数据，不同呈现）
  - 自定义坐标轴：breaks/labels/limits 参数控制器

### m1-p7 ggplot2 图表工作坊
**教学目标**：掌握 12+ 种常用图表类型的参数调节。
- 图表类型网格入口（每种有缩略 SVG + 名称 + 适用数据标签）
- 全屏工作坊（每种图表）：
  - 左面板：专属参数控件（scatter 的形状/大小/抖动，bar 的排列/柱宽/误差线…）
  - 中面板：SVG 实时预览（深色画布 + 坐标轴 + 数据点）
  - 右面板：R 代码（CodeMirror，语法高亮 + 可编辑 + 复制）
  - 底部：一键导出完整 R 脚本
- 支持图表类型：scatter / bar / line / boxplot / violin / histogram / density / heatmap / area / errorbar / lollipop / ridgeline
- **移动端**：三面板改为垂直堆叠 + 手风琴折叠切换

### m1-p8 R 配色与出版级图表
**教学目标**：掌握 R 中专业配色包的使用，将图表调整到顶刊投稿标准。
- **R 配色系统**：
  - 配色包浏览器：RColorBrewer / viridis / ggsci / wesanderson / MetBrewer
  - 每个包的全系列色板展示 + 点击预览应用到图表
  - scale_color vs scale_fill 使用场景
  - 自定义配色函数教学（scale_color_manual / scale_color_gradient / scale_color_gradientn）
- **出版级调整**：
  - 主题定制器：字号/颜色/网格线/背景 全可调 → 实时 SVG 预览 + theme() 代码
  - 期刊规格表（Nature/Science/Cell/PNAS/Lancet）：尺寸/DPI/格式/字体要求
  - ggsave 代码生成器：选期刊+栏宽+格式 → 生成完整导出代码
  - patchwork 多面板布局（4 种布局预览：A|B, A/B, (A|B)/C, (A+B)/(C+D)）

### m1-p9 Python 可视化与数据叙事
**教学目标**：用 matplotlib/seaborn 制作出版级图表，让图表自己"讲故事"。
- **Python 可视化**：
  - matplotlib 基础：Figure/Axes/Artist 层次结构交互图解
  - seaborn 速查：10 种常用图表的参数面板 + 代码生成
  - matplotlib vs ggplot2 对照表（相同图表的两种代码并排对比）
  - 美化：rcParams 全局设置 + 自定义样式表
- **数据叙事与标注**：
  - 标注技巧（annotate 交互演示：箭头/文字/框/高亮区域，拖拽定位）
  - 图表叙事结构：引导读者视线的 4 种方法
    - 颜色对比突出关键数据
    - 标注引导阅读顺序
    - 灰化非重点数据
    - 分步揭示（动画演示）
  - 好/差标注对比

### m1-p10 科研绘图工作流与导出
**教学目标**：掌握从数据到出版级图表的完整流程和正确导出参数。
- **完整工作流**：
  - 全流程时间线（粘性滚动教学）：数据整理 → 选图表 → 初稿 → 配色 → 标注 → 导出
  - Nature 论文 Figure 制作案例（6 步，每步有截图+代码+陷阱提示）
  - 工具链推荐矩阵（R/Python/AI/Illustrator 适合什么场景）
- **导出与格式**：
  - 格式深度对比（SVG/PDF/PNG/TIFF/EPS）
  - 每种格式的内部原理动画演示（矢量如何描述形状 vs 位图如何存储像素）
  - DPI 效果对比器：同一图表用 72/150/300/600 DPI 并排看差异
  - 分辨率计算器：输入实际尺寸(cm) + DPI → 像素数 + 文件大小估算
  - 期刊要求速查：输入期刊名 → 显示图片规格要求
